* Создание ветки
Для создания ветки используйте команду (вместо "test" укажите название
создаваемой ветки):
#+BEGIN_SRC bash
git branch test
#+END_SRC

После этого вы всё ещё находитесь в предыдущей ветке (скорее всего в
ветке по умолчанию "master")

Чтобы перейти в ветку с названием "test" сделайте
#+BEGIN_SRC bash
git checkout test
#+END_SRC


Следующая команда объединяет две предыдущие (создать ветку и перейти в неё) 
#+BEGIN_SRC bash
git checkout -b test
#+END_SRC


* Просмотр дерева

в терминале
#+BEGIN_SRC bash
git log --oneline --abbrev-commit --all --graph --decorate --color
#+END_SRC

графическая версия (с опцией "--all" показывает все ветви; --date-order сортирует коммиты по времени)
#+BEGIN_SRC bash
gitk --all --date-order
#+END_SRC
